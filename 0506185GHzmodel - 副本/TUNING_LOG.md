# 185g 模型调参记录

基准模型：`diplexer185GHzmodel.aedt`
优化目标：在保留 185g 模型 S11/IL 优势的前提下，将 S31@19GHz 压至 ≤ −10 dB，同时维持交叉频率 ≥ 18.5 GHz。

---

## 基准性能（baseline，as-is）

| 指标 | 值 | 目标 | 状态 |
|---|---|---|---|
| 交叉频率 | 18.504 GHz | ≥ 18.5 GHz | ✓ |
| S31 @ 19 GHz | −7.9 dB | ≤ −10 dB | ✗ |
| S31 @ 20 GHz | −8.1 dB | ≤ −10 dB | ✗ |
| 通带波纹 20–25 GHz | 0.87 dB | ≤ 1 dB | ✓ |
| 最小插损（20 GHz+） | −1.6 dB | — | （优） |
| S11 worst（19–25 GHz） | −7.0 dB | — | （优） |

**与 12g 最优（r27_c1y022）对比：** 185g 的 S11 和 IL 明显更好（S11 −7.0 vs −3.9，IL −1.6 vs −3.1），但 S31 抑制不足，主要原因是电容宽度较窄（`w_C4=1.0mm, w_C2=0.9mm` vs 12g 的 1.45mm）。

关键参数（baseline）：
- `w_C4185g = 1.0 mm`，`w_C2185g = 0.9 mm`（电容宽度）
- `L4_g3185g = 0.85 mm`，`L2_g3185g = 0.85 mm`（LPF stub 长度）
- `l_L5185g = 0.7248 mm`，`l_C4185g = 0.465 mm`，`l_L3185g = 0.936 mm`（LPF 元件长度）

---

## Round 1 — sweep1：w_C4 × w_C2 扫参（2026-05-06）

### 动机
S31 不足的根本原因是电容宽度过窄。加宽电容可增大并联电容量，提升 LPF 阻带抑制，代价是 LPF 截止频率下移（交叉频率下降）。

### 参数设置
- `w_C4185g`：1.05 / 1.15 / 1.25 mm（基线 1.0）
- `w_C2185g`：0.95 / 1.05 / 1.15 mm（基线 0.9）
- 其余固定：`L4_g3185g=0.85`, `L2_g3185g=0.85`, `L7_g3185g=0.9`
- 共 9 个 case（L9 全因子）

### 结果汇总

### 反思
1. **S31 可调性良好**：加宽 wC4 0.1mm → S31@19 约改善 1.5–2.0 dB，加宽 wC2 0.1mm → 约改善 1.0 dB。
2. **关键矛盾**：S31 达标所需的最小电容宽度（wC4≥1.05, wC2≥1.05）会使交叉频率下降约 120 MHz（18.381 vs 18.5），不满足目标。
3. **IL/S11 完全不受影响**：所有 case 的 IL 和 S11 与基线几乎一致，这意味着加宽电容不会破坏 HPF 通带质量。
4. **最佳候选**：`wC4=1.05, wC2=1.05`——S31 刚好达标，交叉仅差 119 MHz，是 Phase 2 的起点。

### 下一步
Phase 2：在 `wC4=1.05, wC2=1.05` 基础上，用 k 因子整体缩短 LPF 元件长度，把交叉频率从 18.381 拉回到 ≥ 18.5 GHz。
- k 范围：0.94 ~ 0.99（类比 12g 模型中 k=0.90 拉升了约 300 MHz）
- 预期：k≈0.96–0.97 可补回 119 MHz 缺口

---

## Round 2 — refine1：k 因子扫参（2026-05-06）

### 动机
002（wC4=1.05, wC2=1.05）S31 达标但 crossing 差 119 MHz。用 k 缩短 LPF 元件长度来拉升 crossing，类比 12g 中 k=0.90 的作用。

### 参数设置
- 基础：wC4=1.05, wC2=1.05（002 参数）
- `k`：0.94 / 0.95 / 0.96 / 0.97 / 0.98，LPF 5 个元件等比缩放

### 结果汇总

| UID | k | 交叉 GHz | S31@19 dB | 备注 |
|---|---|---|---|---|
| 010 | 0.94 | 18.583 ✓ | −7.1 ✗ | crossing 充裕，S31 大幅退化 |
| 011 | 0.95 | 18.585 ✓ | −7.4 ✗ | |
| 012 | 0.96 | 18.524 ✓ | −8.0 ✗ | crossing 达标，S31 差 2 dB |
| 013 | 0.97 | 18.476 ✗ | −8.8 ✗ | |
| 014 | 0.98 | 18.420 ✗ | −9.2 ✗ | |

### 反思
k 减小同时让 crossing 上升和 S31 变差，两者完全耦合。k 单独无法同时满足两个目标。

## Round 3 — refine2：k × wC4 二维扫参（2026-05-06）

### 动机
k=0.96（012）crossing 达标，但 S31 差 2 dB。增大 wC4 可补 S31，但也会压低 crossing。寻找平衡点。

### 参数设置
- `k`：0.96, 0.97
- `wC4`：1.15, 1.20, 1.25, 1.30
- `wC2`：固定 1.05

### 结果汇总

| UID | k | wC4 | 交叉 GHz | S31@19 dB | 备注 |
|---|---|---|---|---|---|
| 015 | 0.96 | 1.15 | 18.372 ✗ | −9.8 ✗ | **最接近**：crossing 差 128 MHz，S31 差 0.2 dB |
| 016 | 0.96 | 1.20 | 18.208 ✗ | −11.1 ✓ | HPF S11 第一峰抬至 ~−5 dB → 排除 |
| 019 | 0.97 | 1.15 | 18.348 ✗ | −10.6 ✓ | crossing 差 152 MHz |

### 反思
1. **wC4 ≤ 1.15mm 是硬上限**：wC4=1.20 导致 HPF S11 第一波峰抬至约 −5 dB，通带恶化。
2. k × wC4 空间内无可行点：wC4 每加宽 0.1mm，crossing 掉约 150 MHz，但 S31 只改善 1.5 dB，斜率不利。
3. **015 是最接近点**（差 128 MHz crossing + 差 0.2 dB S31），需借助其他参数突破。

## Round 4 — refine3：L2_g3185g 扫参（2026-05-06）✓ 找到解

### 动机
k 和 wC4 已锁定（k=0.96, wC4=1.15）。015 的 crossing 差 128 MHz，需要从 **HPF 侧**调参而非 LPF 侧。
- 12g 最优：L2_g312g=0.75mm（比 L4_g312g=0.85 短 0.1mm）
- 185g 基线：L2_g3185g=0.85mm（与 L4_g3 相同）
- 假设：减小 L2_g3185g 可上移 HPF 通带边沿 → crossing 上升

### 参数设置
- 基础：k=0.96, wC4=1.15, wC2=1.05（015 参数）
- `L2_g3185g`：0.75 / 0.78 / 0.81 / 0.84 mm

### 结果汇总

| UID | L2_g3 | 交叉 GHz | S31@19 dB | S31@20 dB | 波纹 dB | IL dB | 状态 |
|---|---|---|---|---|---|---|---|
| 023 | 0.75 | 18.634 ✓ | −9.2 ✗ | −12.7 ✓ | 0.50 ✓ | −1.2 | crossing 富裕，S31@19 稍差 |
| **024** | **0.78** | **18.535 ✓** | **−10.3 ✓** | **−11.9 ✓** | **0.56 ✓** | **−1.3** | **★ 全部达标** |
| 025 | 0.81 | 18.383 ✗ | −11.3 ✓ | −11.0 ✓ | 0.62 ✓ | −1.5 | crossing 差 117 MHz |
| 026 | 0.84 | 18.395 ✗ | −9.9 ✗ | −10.4 ✓ | 0.81 ✓ | −1.5 | |

### 反思
L2_g3185g 减小对 crossing 的敏感度极高（0.75→0.84 覆盖了 239 MHz 范围），且对 S31@20 和波纹的影响是正向的。**0.78mm 是甜点**。

### 优化路径总结
1. 加宽 wC4/wC2 → S31 改善但 crossing 下降（方向确认）
2. k 因子缩短 LPF → crossing 回升但 S31 退化（耦合矛盾）
3. **突破：减小 L2_g3185g（0.85→0.78）→ HPF 通带边沿上移，crossing 独立上升，S31 保持**
